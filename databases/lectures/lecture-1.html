¿Qué es una
base de datos?

Una base de datos (DB, en adelante) es un sistema computarizado de almacenamiento de registros

¿Para qué sirve una
base de datos?

Una base de datos sirve para efectuar operaciones sobre los registros de una forma estándar y ordenada

¿Qué operaciones se pueden
efectuar sobre los registros?

Las operaciones básicas son:
 * consultar registros
 * agregar registros
 * actualizar registros
 * borrar registros

 
SELECT
INSERT
UPDATE
DELETE

¿Porqué usar
bases de datos?

Usar bases de datos para almacenar registros presenta ventajas como...

Los datos pueden ser compartidos entre usuarios y aplicaciones de distinta índole.

Se elimina la redundancia al evitar guardar datos repetidos entre distintos usuarios o aplicaciones.

Se elimina la inconsistencia al asegurarse que todos los datos de un mismo tipo contengan los mismos campos.

Provee soporte para transacciones asegurando que los datos se guarden completos, o no se guarde nada en absoluto si ocurre un error.

Asegura la integridad garantizando que los datos que se obtienen de la base son correctos, para todo usuario o aplicación, en cualquier momento.

Ofrece seguridad al proveer sólo métodos autorizados para acceder a los datos.

y lo más importante...

Estandariza la forma de modelar y representar datos, evitando la creación de otro formato de datos personalizado más.
